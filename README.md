# 📚 Moje Maturitní Knihy

Moderní webová aplikace určená pro studenty k přípravě na ústní část maturitní zkoušky z českého jazyka a literatury.

![Aplikace Preview](https://github.com/Datlik/Maturitni-knizky/raw/main/preview.png) *(Poznámka: Přidejte vlastní screenshot)*

## ✨ Funkce

- **Kompletní seznam děl**: Přehled 20 vybraných děl české i světové literatury.
- **Interaktivní kvíz**: Testování znalostí (10 otázek v kole) s okamžitou zpětnou vazbou.
- **Chytré vyhledávání**: Full-textové vyhledávání podle autora nebo názvu díla s AND logikou a zvýrazněním.
- **Systém oblíbených**: Označte si knihy hvězdičkou a zaměřte na ně svůj studijní plán nebo kvíz.
- **Filtrování podle období**: Rychlé řazení děl do historických epoch (I-IV).
- **Odpočet k maturitě**: Neustálý přehled o tom, kolik času zbývá do 25. 5. 2026.
- **Dark Mode**: Příjemné prostředí pro noční studium.
- **Moderní UI**: Responzivní design, plynulé animace a čisté rozhraní.

## 🚀 Jak aplikaci spustit

Aplikace je postavena na čistém HTML, CSS a JavaScriptu (Vanilla JS), takže nevyžaduje žádnou instalaci:

1. Stáhněte si repozitář nebo jej naklonujte.
2. Otevřete soubor `index.html` v libovolném moderním prohlížeči.

## 🛠️ Technologie

- **Struktura**: HTML5 (Sémantické prvky)
- **Styling**: CSS3 (Custom properties, Flexbox, Grid, Keyframe animations)
- **Logika**: JavaScript (ES6+, LocalStorage pro ukládání oblíbených a barevného schématu)
